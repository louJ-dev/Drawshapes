# Drawshapes
  algorithms for printing basic shapes using c-sharp
  
## Included Shapes:
- Line
- Striped Line
- Square
- Parallelogram
- Triangle
- Reverse Triangle
- Isoceles Triangle
- Reverse Isoceles Triangle
- HourGlass
- Diamond
- Zero
- Up arrow
- Down Arrow
- X
- Bow Tie

> [!NOTE]
>   Code was test using size 7 and can be change by changing _size_ variable in Program.cs
